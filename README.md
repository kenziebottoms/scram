# Instagram scraping

## Endpoints

| Path              | Method | Result                  |
| ----------------- | ------ | ----------------------- |
| `users/:username/about` | `GET`  | Get user metadata |
| `users/:username/posts` | `GET` | Get user's three most recent posts |
